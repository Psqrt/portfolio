R Flexdashboard - MyAnimeList dataset full interactive dashboard

Flexdashboard est un outil sur R qui permet de construire une interface (dashboard) qui peut être totalement interactive sans avoir besoin d'une session R qui tourne (ce qui n'est pas le cas des applications Shiny). Le dashboard peut alors être consulté sur un appareil sans R et sans internet.

Ce flexdashboard traite la base de données du site MyAnimeList, qui est l'équivalent du site IMDb mais pour les animés et films d'animation. (d'ailleurs, pour avoir traité la base de données de ces deux sites auparavant, la structure est très similaire et donc ce dashboard peut être facilement adapté pour accueillir la base d'IMDb).

Modélisation économique - Discrimination salariale entre H/F

La question de l'inégalité salariale revient souvent dans les médias. Cependant, les chiffres avancés ne s'accordent pas et sont très souvent mal interprétés. La discrimination ne s'arrête pas à une comparaison du salaire moyen entre les deux groupes. Des facteurs tels que les différences de postes occupés peuvent augmenter l'écart moyen de rémunération entre les deux groupes. Ceci n'est pas amputable à la discrimination de rémunération.

La méthode de décomposition Oaxaca-Blinder, ici utilisée, permet de quantifier l'écart moyen de salaires pour des groupes homogènes (ayant les mêmes caractéristiques individuelles et occupant les mêmes postes au travail).
